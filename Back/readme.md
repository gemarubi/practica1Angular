Iniciar Back:
Desde la carpeta back:
    npm install
    nodemon

Iniciar Front
Desde la carpeta front:
    npm install
    npm start


Migraciones:
    migrar tablas:
    npx sequelize-cli db:migrate
    ejecutar seeder:
    npx sequelize-cli db:seed:all
